# transform


## origin

> transform-origin `相对于当前设置该属性的元素的位置`

## rotate
> 旋转的时候，元素相对的`坐标轴也会跟着一起旋转`。注意旋转的`顺序问题`

**2d**
> rotateX(angle)(angle) | rotateZ(angle) 

**3d**

> rotate3D(x,y,z,angle) x,y,z(`3个向量`) 确定在三维坐标轴上面的旋转轴
